# metro_management_system
Database design and ERD for Metro Management System

## METRO OVERVIEW
----------
This prject focus on designing a relational database for a **Metro Management System** and write some functions related to Metro database to get information. 
It includes ERD design, relational schema and SQL scripts 
Notes: the dataset is formed automatically
